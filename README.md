## Rain

A smarter weather app powered by Electron.

To run application:
> Open Terminal or Command Prompt with Node.js Installed
>
> `cd ../rain`
> 
> (if `node_modules` are not installed) `npm i`
> 
> `npm start`

[Download Latest Node.js and NPM Here](https://nodejs.org/en/)
